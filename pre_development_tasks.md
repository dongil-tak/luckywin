## 2. 🔌 외부 API 연동 계정 및 라이선스 확보
개발팀이 연동 작업을 즉시 시작할 수 있도록 API 키 및 인증 수단을 확보해야 합니다.

*   [ ] **공공데이터포털 복권 판매점 API 인증키 발급**
    *   **신청처**: [공공데이터포털(data.go.kr)](https://www.data.go.kr)
    *   **대상 API**: 기획재정부 복권위원회 로또복권 판매소 정보 API.
*   [ ] **지도 API 개발자 계정 생성 및 SDK 키 발급**
    *   **선택 플랫폼**: 카카오 개발자 센터 (Kakao Developers) 또는 네이버 클라우드 플랫폼 (NAVER Cloud Platform).
    *   **역할**: `LottoStore.tsx` 화면에서 내 주변 판매점 위치 지도 매핑 및 마커 표시용.
*   [ ] **공식 당첨결과 동기화 데이터 파이프라인 준비**
    *   동행복권 당첨 결과를 매주 토요일 오후 8시 45분 이후 자동으로 가져와서 데이터베이스(혹은 JSON 데이터)를 업데이트할 수 있는 스케줄러(Cron Job) 설계 합의.
